Project Plan 2022

Objective / Purpose

To create two websites - arltechCoop, a website for the worker cooperative at ACC / Arlington Tech, and acclib, a web application cataloging my classroom book collection.

Background

The goal of the web development pathway at the Arlington Career Center is to provide an engaging, mutually supportive learning environment for the attainment of the knowledge and skills needed to be an entry level web developer.

Specific Aims

This project will create two model websites using the web technologies on which our educational pathway will focus - HTML, CSS, JavaScript, PHP, and Python.

Resources / Methods

All software used in the projects will be FOSS, and OER materials will be used whenever possible. Since students in the first course in our pathway will be taking the CIW Advanced HTML5 & CSS3 Specialist certification, I will be integrating and adapting the curriculum materials and labs into this project as well.
